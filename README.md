Sweet

Descripción: Es ek front-end de una aplicación movil, la cual es la creación de un recetario de postrería

### Tecnologías Utilizadas
Lenguaje: TypeScript
Framework:React Native
Herramientas de Desarrollo: Expo Go
Gestión de Dependencias: npm

Para ejecutar este proyecto de forma local, sigue estos pasos
1. Clonar el repositorio:
   ```bash
   git clone [https://github.com/tu-usuario/sweet.git](https://github.com/tu-usuario/sweet.git)
  2. Instalar dependencias:
     npm install
  3. Ejecutar la aplicación:
     npx expo start
  4. Visualización:
Descarga la app Expo Go en tu dispositivo móvil (iOS o Android) y escanea el código QR que aparecerá en tu terminal
